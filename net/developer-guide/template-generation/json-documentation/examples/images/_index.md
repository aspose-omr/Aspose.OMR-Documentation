## **Create OMR Templates with images**
To add the image, add the following to the json markup with the name of the image

````json
{
  "children": [
    {
      "children": [
        {		
			"align": "Center",
			"name": "Aspose.jpg",
			"element_type": "Image"
		}
      ],
      "element_type": "Page"
    }
  ],
  "element_type": "Template"
}
````
