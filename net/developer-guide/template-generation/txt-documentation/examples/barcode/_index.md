## **Create OMR Templates with Barcodes**
Aspose.OMR for .NET provides the option to generate OMR template (.omr)Â files containing barcodes. For this, you need to add the following part in the text markup.

```text
?barcode=AsposeWebsite
	value=aspose.com
	barcode_type=qr
	qr_version=1
	x=2000
	y=120
	height=360
```



### Full template content

	barcode_type=qr
	qr_version=1
